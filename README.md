# Zomato-Dataset-Analysis
Zomato Restaurant Data Analysis using Python, Pandas, Matplotlib, Seaborn, and WordCloud.
## Objective
Analyze restaurant and review data to extract insights on ratings, cuisines, locations, and factors affecting ratings.
## Dataset
Zomato Restaurant Dataset from Kaggle.
## Tools Used
1.Python
2.Pandas
3.NumPy
4.Matplotlib
5.Seaborn
6.WordCloud
7.Google Colab
## Analysis Performed
1.Data Cleaning
2.Missing value handling
3.Duplicate Removal
4.Rating Analysis
5.Location Analysis
6.Cuisine Analysis
7.Correlation Analysis
8.Heatmap Visualization
9.Word Cloud Visualization
## Recommendations
1. Partner with highly rated restaurants.
2. Promote popular cuisines.
3. Focus marketing on hotspot locations.
4. Improve recommendation systems using ratings and votes.
5. Highlight affordable restaurants with high ratings.
